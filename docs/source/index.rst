太初平台中文文档
===================================

太初是一个分布式多智能体强化学习平台。太初提供灵活的模块化算法实现，其模块化的组件能满足各种多智能体强化学习算法的需求。平台目前包含的算法有：

* COMA
* QMIX

.. note::

   太初目前尚处于开发阶段，后续将加入更多算法

基于python multiprocessing模块搭建的分布式并行系统没有依赖现有分布式平台，因此具有更强的可读性，更高的灵活性和可自定义性。
